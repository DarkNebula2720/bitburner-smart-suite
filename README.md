# Bitburner Smart Suite

An advanced modular framework for Bitburner automation with: 
- Terminal-safe UI
- Smart daemon controller
- Extensible modules for hacking, factions, augments, and more.

## Install

Paste this in the Bitburner terminal:
```
wget https://raw.githubusercontent.com/DarkNebula2720/bitburner-smart-suite/main/install.js install.js && run install.js
```

Then run:
 ```
run daemon.js
 ```
